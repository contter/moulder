# Introduction

![Moulder Framework](/img/banner.png)

## What is Moulder?

This is a Swiss knife for creating artwork.

## Features {#features}

Moulder is built with high attention to the developer and creator experience.

- **🎛️ Pluggable**:
  - Customizable components.
- **🧶️ Editor**:
  - Node-based editor out of the box.
- **⚛️ Built with React**:
  - Extend and customize with React.
- **🖊️ Render any libraries**:
  - Using any js library (p5js, threejs and etc).
- **📸 Capture**:
  - Export to images, svg and other.
